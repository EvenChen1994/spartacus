### SNAT ports limit

| Pool size (VM instances) | Default SNAT ports |
| ------------------------ | ------------------ |
| 1-50                     | 1024               |
| 51-100                   | 512                |
| 101-200                  | 256                |
| 201-400                  | 128                |
| 401-800                  | 64                 |
| 801-1000                 | 32                 |

[source](https://learn.microsoft.com/en-us/azure/load-balancer/load-balancer-outbound-connections#preallocatedports)
